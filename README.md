# Ultrasonic Distance Sensor Project

## 📌 Description
This project is based on an HC-SR04 ultrasonic sensor with Arduino to measure distance between the sensor and an object.

## ⚙️ How it Works
The sensor sends ultrasonic waves and receives the reflected signal. The time taken for the echo is used to calculate distance.

## 📏 Formula
Distance = (Time × Speed of Sound) / 2

## 🛠 Components Used
- Arduino Uno
- HC-SR04 Sensor
- Jumper Wires

## 💡 Applications
- Obstacle detection
- Robotics
- Smart parking systems
